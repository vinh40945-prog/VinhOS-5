# BACKGROUND #

có thể thay đổi background chỉ cần kéo thả ảnh hoạc video vào hoặc ấn import để chọn ảnh hoặc video

*lưu ý: Background bằng video vẫn chưa được tối ưu sẽ gây ra giật lag*

# EDIT MODE #

trong phần nhập văn bản có thể paste văn bản hoặc nhập thủ công. Sau khi nhập văn bản thì ấn dấu cách để tiếp tục nhập thêm.

Có thể thêm hiệu ứng khi trình chiếu

**Phím tắt**

SPACE `(để tiếp tục nhập chữ)`

ENTER `(hoàn thành)`

BKSP `(dùng để xoá văn bản khi cần chỉnh sửa)`

CTRL + A `(xoá toàn bộ)`

Click `(để chọn chữ muốn sửa hoặc thêm hiệu ứng)`

CTRL + Click `(bắt buộc phải chọn từ 2 trở lên để group)`

*lưu ý: khi group vẫn sẽ chiếu như bình thường chỉ là các vật đã group sẽ không cách nhau*

# START #

sau khi đã chỉnh sửa và thêm hiệu ứng. Ấn esc để quay lại menu chính và ấn start để bắt đầu trình chiếu.

**Phím tắt**

SPACE `(để trình chiếu)`

BKSP `(Trình chiếu lại)`

CTRL + A `(clear tạm thời)`

*lưu ý: nó chỉ xoá chữ tạm thời có trên màn hình, vẫn có thể ấn SPACE để tiếp tục*

ESC `để trở về menu`
